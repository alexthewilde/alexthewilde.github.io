Behat

1. describe the behavior of a feature in plain language

Feature: display a personal user login messages
  In order to personalize the user experience
  As a user when I'm logging into the site
  I get a personal welcome message

  Scenario: Login with valid credentials
    Given I login as "alex" with "secret"
    Then I should see the message "Welcome back alex!"

2. write step definitions using SimpleTest API

<?php
$steps->Given('/^I login as "([^"]*)" with "([^"]*)"$/', function($world, $name, $pass) {
  $world->drupalPost('user', array('name' => $name, 'pass' => $pass), t('Log in'));
});

$steps->Then('/^I should see the message "([^"]*)"$/', function($world, $message) {
  $world->assertText($message);
});

3. run and watch it fail

$ behat sites/all/modules/custom_login/features

4. write code to make the step pass

<?php
/**
 * Implementation of hook_user_login().
 */
function custom_login_user_login(&$edit, $account) {
  drupal_set_message(t('Welcome back %name!', array('%name' => $account->name)));
}
